## Overview
In this roadmap, you will find the features that are being prioritized and the new capabilities we plan to develop for the `melper` library. We are committed to expanding the library with additional utility functions that help developers save time and enhance the quality of their code.

## Upcoming Features

<Info>While the developments listed are our current priorities, we are also planning to introduce new modules and functions to further enhance the `melper` library. Stay tuned for continuous updates and new features.</Info>

### Type Module
- [x] `lookup`: Check the type of a variable accurately.
- [ ] `isNull`: Determine whether a variable is null.
- [ ] `isBoolean`: Check if a variable is a Boolean value.
- [ ] `isBuffer`: Verify if a variable is a buffer.
- [ ] `isNumber`: Confirm if a variable is of type number.
- [ ] `isString`: Confirm if a variable is of type string.
- [ ] `isArguments`: Determine if a variable is an arguments object.
- [ ] `isObject`: Check if a variable is a generic object.
- [ ] `isDate`: Verify if a variable is a date object.
- [ ] `isArray`: Determine if a variable is an array.
- [ ] `isRegexp`: Check if a variable is a regular expression.
- [ ] `isError`: Determine if a variable is an error object.
- [ ] `isFunction`: Check if a variable is a function.
- [ ] `isClass`: Check if a variable is an instance of a class.
- [ ] `isInteger`: Determine if a variable is an integer.
- [ ] `isFloat`: Check if a variable is a floating-point number.
- [ ] `isDecimal`: Determine if a variable is a decimal number.

Currently, we have implemented the `lookup` function in the `type` module, which allows users to easily check the type of a variable. We plan to expand this module with more specific and useful type-checking functions.

---

### String Module
- [ ] `slug`: Convert strings into URL-friendly slugs, replacing spaces with dashes and lowercasing all characters.
- [ ] `random`: Generate a random string of specified length, optionally including numbers and symbols.
- [ ] `excerpt`: Extract a brief snippet from a longer text, ideally for previews or summaries.
- [ ] `truncate`: Shorten a string to a specified length and append ellipses or other characters if it exceeds that length.
- [ ] `interpolate`: Perform template string interpolation, replacing placeholders with provided values in a string template.

We are developing new functions for string manipulation

## Conclusion
As we continue to develop `melper`, your feedback and contributions are invaluable to us. We encourage the community to engage with the development process, propose new features, and provide feedback on existing functionalities. Together, we can make `melper` even more powerful and useful for all developers. Stay connected and watch for more updates as we grow and evolve.
